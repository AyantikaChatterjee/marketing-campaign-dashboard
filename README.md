# 📊 Global Advertising Performance Dashboard

## Overview
This project analyzes cross-platform advertising performance across Google Ads, Meta Ads, and TikTok Ads.

The goal is to understand how ad spend translates into revenue and identify opportunities to optimize campaign performance and improve ROAS.

---

## Key Insights
- Google Ads delivers the highest revenue and strongest ROAS  
- TikTok Ads drives volume but lower efficiency  
- Search campaigns show the highest conversion intent  
- Budget can be optimized by focusing on high-performing channels
